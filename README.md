# navbytes/homebrew-tap

Homebrew tap for [Vee](https://github.com/navbytes/vee), a native macOS menu-bar
script runner.

```sh
brew tap navbytes/tap
brew install --cask vee
```

`brew upgrade --cask vee` picks up new releases automatically.

## Notes

- Requirements match the app: **Apple Silicon** and **macOS 26 (Tahoe) or later**.
- `Casks/vee.rb` is generated by Vee's release workflow
  ([`scripts/update-homebrew-cask.sh`](https://github.com/navbytes/vee/blob/main/scripts/update-homebrew-cask.sh))
  on every release — edit that script, not the cask here.
